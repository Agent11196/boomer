# Boomer

Zoomer application for boomers

- Development is done on https://twitch.tv/tsoding
- Archive of the streams: https://www.twitch.tv/collections/HlRy-q69uBXmpQ

## Quick Start

```console
$ nix-shell   # only for NixOS
$ nimble build
$ ./boomer
```
